# 健身问答知识库（基础版）

## 1. 初学者一周练几次比较好？

- 建议 **每周 3-4 次**，先建立稳定习惯，再逐步增加训练量。
- 每次训练建议包含：**热身 → 主训练 → 拉伸/放松**。

## 2. 减脂应该先有氧还是先力量？

- 多数情况下，优先保证 **力量训练质量**，再做中等强度有氧。
- 如果时间有限：力量 + 少量有氧，长期更易坚持。

## 3. 深蹲膝盖内扣怎么办？

- 先降重量/降速度，确保脚掌稳定（脚跟不抬起）。
- 重点关注：髋外旋/臀中肌发力、膝盖方向与脚尖方向一致。

## 4. 俯卧撑塌腰怎么办？

- 降阶（跪姿俯卧撑/抬高手俯卧撑），先练核心稳定。
- 提示：收紧腹部、夹臀、肩胛稳定（避免耸肩）。

## 5. 训练后肌肉酸痛正常吗？

- 轻到中度酸痛常见（尤其是新动作/新强度），通常 24-72 小时缓解。
- 若出现锐痛、关节痛、麻木或持续加重，应停止并评估。

## 6. 如何避免受伤？

- 动作质量优先于强度；循序渐进；保证睡眠与恢复。
- 训练过程中出现疼痛/头晕/胸闷应立即停止。
