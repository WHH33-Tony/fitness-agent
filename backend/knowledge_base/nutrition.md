# 营养知识库（基础版）

## 1. 三大宏量营养素

- **蛋白质**：用于肌肉修复与合成。力量训练/增肌期更重要。
- **碳水化合物**：训练的主要能量来源，影响训练表现与恢复。
- **脂肪**：参与激素合成、维生素吸收；不建议长期极低脂饮食。

## 2. 蛋白质摄入建议（常用经验范围）

> 以“体重(kg)”为基准估算每日蛋白质总量，实际需结合训练量、体脂、肾脏健康情况。

- **日常健康/轻度运动**：\(0.8 \sim 1.2\) g/kg/天
- **减脂期（保肌）**：\(1.6 \sim 2.2\) g/kg/天
- **增肌期/力量训练频繁**：\(1.6 \sim 2.4\) g/kg/天

## 3. 训练前后怎么吃（简化版）

- **训练前 1-2 小时**：优先“碳水 + 少量蛋白”，避免过油过辣。
- **训练后 0-2 小时**：补充“蛋白 + 碳水”有利于恢复；总量更重要，不必过度追求“黄金窗口”。

## 4. 常见食物参考（概念性示例）

- **鸡胸肉**：高蛋白、低脂
- **鸡蛋**：蛋白质质量高，但注意烹饪方式与总脂肪摄入
- **牛奶/酸奶**：补蛋白同时带碳水和脂肪，适合作为加餐
- **米饭/面/土豆/燕麦**：常见碳水来源
- **坚果/橄榄油**：脂肪密度高，适量即可

## 5. 注意事项

- 有**肾脏疾病**、痛风等情况，蛋白质摄入需遵医嘱。
- 体重变化的关键仍是**长期能量收支**与可持续执行。
